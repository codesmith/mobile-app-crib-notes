アプリ開発では、コーディングルールを定めることでコードの品質を保つことができます。

このアプリでは、必ず守るべきルールと推奨するルールの2つを定めています。

## 自動でチェック可能なルール

ESLintやPrettier、TypeScriptなどの静的解析ツールを導入します。
